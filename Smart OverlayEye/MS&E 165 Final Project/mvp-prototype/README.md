# Pear AI Step Navigator — MVP Prototype

Interactive tablet guide prototype for the MS&E 165 CPI final project.

## Run

Open `index.html` in a browser. No build step required.

For tablet-like testing: resize browser to ~768×1024 or use DevTools device emulation.

## Tasks (hardcoded)

- **Remove background** (Photoshop) — 4 steps
- **Color grade photo** (Lightroom) — 4 steps  
- **Export for web** (Figma) — 4 steps

## Flow

1. User selects a task
2. Steps appear one-by-one; user clicks "Next step"
3. Completion screen with "Start over"
